Parameters

node_type
conn_mode连接模式，当node_type不为0的时候是必须 取值范围： 0表示直连模式、1表示流媒体服务器模式、 2表示P2P云模式、3表示云流媒体模式； （The connection mode, when node_type is not 0 to range: 0 direct mode, 1 streaming media server mode, 2 P2P cloud model, 3 media stream cloud model;）
vendor_id厂商ID，当node_type不为0的时候是必须的，取值范围： 1001表示华科流媒体协议（bit）、1002表示华科流媒体协议（XML）、 1003表示华科流媒体协议（JSON）、1004表示华科流媒体协议（OWSP）、 1005表示原华科流媒体服务器协议（HKSP）、1006表示文件摄像机协议（FCAM）、 1007表示华科监控通讯协议（HMCP）、1008表示华科设备上连协议（HDTP）、 1009表示UMSP、1010表示EPMY、1011表示RTSP、 1012表示HTTP、1013表示SIP、1014表示RTMP、 2010表示杭州海康、2011表示杭州海康推模式、2020表示杭州大华、 2030表示深圳锐明、2040表示深圳黄河、2050表示深圳汉邦、 2060表示杭州雄迈、2070表示中山立堡、2080表示成都索贝、 2090表示上海皓维、2100表示金三立、2110表示上海通立、 2120表示深圳郎驰、2130表示网视通、2140表示广州奇幻、 2150表示安联锐视、2160表示广州佳可、2170表示深圳旗翰、 2180表示瀚晖威视。
dev_umid设备UMID，当node_type不为0是必须的，最大长度为24个字符；（Device UMID，）
dev_addr设备地址，当node_type不为0是必须的，最大长度为64个字符；
dev_port设备端口，当node_type不为0是必须的，端口范围为1到65535；
dev_user设备用户名，当node_type不为0的时候是必须的，最大长度为32个字符；
dev_passwd设备用户密码，当node_type不为0的时候是必须的，最大长度为32个字符；
dev_ch_num设备通道数，当node_type不为0的时候是必须的；
dev_ch_no设备通道号，取值从0开始，当node_type不为0的时候是必须的；
dev_stream_no设备码流号，当node_type不为0的时候是必须的， 取值范围：0表示主码流、1表示子码流、2、3、4、等等。
